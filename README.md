Use this program to check for and dump any shared user data (FCT/Factory) titles on your console. These contain the default Wii U Menu layout if there are any non-system factory preinstalled titles. This can include titles from console bundles (e.g Mario Kart 8, Super Mario Maker, etc.) or other preinstalled software that many Wii Us came with (e.g YouTube, Hulu, various Japanese apps, etc.)

Please zip any files extracted in the FCTCheck-Dump directory created on the root of the SD card and DM them to me (spletz_) on Discord, or contribute to the [title database on WiiUBrew](https://wiiubrew.org/wiki/Title_database#0005000B:_Shared_User_Data_(FCT/Factory)_Titles) yourself. Even if a title already exists on the website, please do not hesitate to send me anything you retrieve as the title could have a different SDK version and/or other information, which is still valuable.
